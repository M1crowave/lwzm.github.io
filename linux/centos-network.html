CentOS 的网络配置其实很简单, ip, mask, gateway, dns 等等都可以在一个地方搞定: /etc/sysconfig/network-scripts/ifcfg-XXX 参考[这篇文档](http://www.centos.org/docs/2/rhl-rg-en-7.2/ch-networkscripts.html) 一般我们配置 `ifcfg-eth0`, 如下: DEVICE="eth0" IPADDR=192.168.0.2 NETMASK=255.255.255.0 GATEWAY=192.168.0.1 DNS1=114.114.114.114 DNS2=8.8.8.8 BROADCAST, NETWORK 可以不配, 系统自己可以算出来. 不重启系统即可应用新的配置: service network restart
