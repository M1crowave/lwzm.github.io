locale 有时会有问题，编辑 /etc/default/locale： 

LANG="en_US.UTF-8"
LANGUAGE="en_US.UTF-8"
LC_ALL="en_US.UTF-8"

使用 stable 软件包

echo 'deb http://ftp.debian.org/debian stable main' >>/etc/apt/sources.list
apt update
apt -t stable install XXX...
