开启 UDP 收日志，以 JSON 格式解码原始日志，输出到控制台： 

processors:
- decode_json_fields:
    fields:
    - message

filebeat.inputs:
- type: udp
  host: :8000

output.console:
  pretty: true

开启 UDP 收日志，清理自带的一些字段，以 JSON 格式解码原始日志到顶层，清理原始日志，输出到控制台： 

processors:

- drop_fields:
    fields:
    - input
    - agent
    - host
    - ecs
    - log

- decode_json_fields:
    target: ""
    fields:
    - message

- drop_fields:
    fields:
    - message


filebeat.inputs:
- type: udp
  host: :8000


output.console:
  pretty: true
